Project Overview

This project was developed as part of a Front-End Development Internship task. The website showcases DevelopersHub's services, portfolio, and provides a professional interface for potential clients to connect with the team.

Features

- **6 Complete Pages** - Home, About, Services, Portfolio, Contact, Meeting Scheduler
- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean design with smooth animations and gradients
- **Functional Contact Form** - Lead generation form with validation
- **Meeting Booking System** - Multi-step scheduler for client consultations
- **Portfolio Filtering** - Interactive project gallery by category
- **SEO-Friendly Structure** - Semantic HTML and meta tags
- **Smooth Navigation** - Fixed navbar with active link highlighting
